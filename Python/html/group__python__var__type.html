def python_base.p_string_operator_fun ( )

字符串操作

  • Python的字符串有2中取值顺序
    • 从左到右索引默认0开始的，最大范围是字符串长度少1
    • 从右到左索引默认从-1开始的， 最大范围是字符串开头
